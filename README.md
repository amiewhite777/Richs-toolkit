# Rich's Toolkit

A mobile-first app for heritage building work in Bath, designed for builders working on Georgian renovations.

## Features

- **Calculators** — Plaster, paint, timber, tiles, concrete, stud walls
- **Snagging Lists** — Room-by-room snag lists with priorities
- **Time & Expenses** — Log hours, receipts, mileage (45p/mile HMRC rate)
- **Suppliers** — Quick-dial contacts with material list builder
- **Weather** — 7-day forecast with work condition assessments
- **Conversions** — Imperial ↔ Metric with Georgian reference dimensions
- **Bath Heritage** — Georgian proportions, Bath stone guide, listed building rules

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

## Deploy

This project works great with:
- **Vercel** — Just connect your GitHub repo
- **Netlify** — Same, connect and deploy
- **GitHub Pages** — Run `npm run build` and deploy the `dist` folder

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
